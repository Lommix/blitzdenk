About Blitzdenk

Blitzdenk is a coding and research harness for POSIX systems. It ships as a small native Zig binary and embeds Lua for configuration and extension.

The design is deliberately direct:

  • terminal UI first
  • provider adapters for OpenAI-style, Anthropic-style, and Ollama/local models
  • tools implemented in Zig, plus user-defined tools in Lua
  • optional SSH mode that routes file and command tools through a remote target
  • MCP and LSP servers exposed as agent tools
  • session history, prompt history, permissions, tasks, subagents, and compaction

Configuration lives in ~/.config/blitzdenk/blitz.lua. A blitz.lua in the current project can override or extend behavior per workspace.

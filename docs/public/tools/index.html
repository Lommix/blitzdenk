Tools

Tools are provider.tool.Tool values: a tool definition plus a Zig or Lua function that returns a ToolResult.

MCP tools

start_mcp starts a configured MCP server by name. src/mcp.zig initializes the server, lists remote tools, prefixes names when configured, and registers each remote tool as a Blitzdenk tool.

LSP tool
